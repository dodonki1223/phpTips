# 文字列の日付から別の日を求めるサンプル集です

## 文字列「yyyy-mm-dd」の形式から明日、昨日、１週間後などの日付計算をした結果を表示します

### 表示結果例

```shell
　対象日:2019-04-15
　　明日:2019-04-16
　　昨日:2019-04-14
　５日後:2019-04-20
　５日前:2019-04-10
１週間後:2019-04-22
１週間前:2019-04-08
　　月初:2019-04-01
　　月末:2019-04-30
　　来月:2019-05-15
　　先月:2019-03-15
　１年後:2020-04-15
　１年前:2018-04-15
```
